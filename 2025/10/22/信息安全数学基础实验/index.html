快速幂

现在存在这么一个问题：

对于aea^eae，当e很大时，计算aaa的nnn次方表示将nnn个aaa乘在一起:an=a×a⋅⋅⋅×a⏟n个a相乘:a^n=\underbrace{a\times a\cdotp\cdotp\cdotp\times a}_{\text{n个a相乘}}:an=n个a相乘a×a⋅⋅⋅×a​​。然而当a,na,na,n太大的时侯，这种方法就不太可行。如何更快更有效的解决这种计算呢？

举个例子：对于3123^{12}312，我们知道：ab+c=ab⋅aca^{b+c}=a^b\cdot a^cab+c=ab⋅ac, a2b=ab⋅ab=(ab)2a^{2b}=a^b\cdot a^b=(a^b)^2a2b=ab⋅ab=(ab)2。如果我们知道363^636，两个363^636相乘36∗36=3123^6*3^6=3^{12}36∗36=312，是不是就可以更好的计算出幂。

二进制取幂的想法是，我们将取幂的任务按照指数的 二进制表示来分割成更小的任务。

素性测试

问题：给你一个自然数n，判定n是否为素数

  1. 初步想法
    我们知道素数的定义是：如果n (n > 1) 是素数，当且仅当n只有因子1和它本身。
    所以根据定义我们就能知道最基本的判断方法：

判断n是否被i整除（i = 2, 3,…, n-1)

复杂度：O(n)O(n)O(n)

如何减少判断次数

我们假设有合数n，那么它可以写成，其中i,j都是大于1的整数，且都是n的因子。
至少有一个因子小于等于n\sqrt{n}n​：反证法证明，假设i，j都大于n\sqrt{n}n​，那么i∗j>n∗n=ni*j>\sqrt{n}*\sqrt{n}=ni∗j>n​∗n​=n这与n=i∗jn=i*jn=i∗j矛盾。

所以我们只需要检查这个2~n\sqrt{n}n​的整数

如果都没有整除n的那么n一定是质数。这是因为任何大于 n 的因子 都会对应一个小于的因子，如果这个小的因子不存在，那么大的因子也不存在。
复杂度：O(n)O(\sqrt n)O(n​)

我们之前都是顺序去排除，有没有更高效的办法？

接下来展示三种筛选方法（转自我之前写的博客）：

 > 朴素筛就是从2开始一直到n，删去所有的倍数，留下的数就是质数。
 
 
过程：从2开始，删去2的倍数；从3开始，删去3的倍数；从4开始删去4的倍数。

我们从上面的过程来看，我们发现筛去2的倍数后，后面筛去4,6,8等的倍数，出现了重复，如何解决？

我们发现，只需要筛去质数2,3,5,7…的倍数，就可以达到筛出所以质数的目的。

为什么可以这样呢？

算术基本定理：

证明：
存在性：
反证法：假设存在大于 1的自然数不能写成素数的乘积，把最小的那个称为 n。
n一定为合数，但每个合数都可以分解成两个严格小于自身而大于 1 的自然数的积。设n=a∗bn=a*bn=a∗b，则根据假设，n是最小的不能被写成素数乘积的自然数，a=p1p2..pia=p_1p_2..p_ia=p1​p2​..pi​，b=q1q2..qib=q_1q_2..q_ib=q1​q2​..qi​，都能被写成素数的乘积，而n=ab=p1p2..piq1q2..qin=ab=p_1p_2..p_iq_1q_2..q_in=ab=p1​p2​..pi​q1​q2​..qi​，也可以写成素数的乘积，由此产生矛盾

唯一性：
欧几里得引理：若素数 p∣abp \mid {ab}p∣ab ,则不是 p∣ap \mid ap∣a ,就是 p∣bp \mid bp∣b 。
首先 nnn 不是质数。将 nnn 用两种方法写出： n=p1p2p3⋯pr=q1q2q3⋯qsn = {p}_{1}{p}_{2}{p}_{3}\cdots {p}_{r} = {q}_{1}{q}_{2}{q}_{3}\cdots {q}_{s}n=p1​p2​p3​⋯pr​=q1​q2​q3​⋯qs​ 。根据引理,质数 p1∣q1q2q3⋯qs{p}_{1} \mid {q}_{1}{q}_{2}{q}_{3}\cdots {q}_{s}p1​∣q1​q2​q3​⋯qs​ ,所以q1,q2,q3⋯qs{q}_{1},{q}_{2},{q}_{3}\cdots {q}_{s}q1​,q2​,q3​⋯qs​ 中有一个能被 p1{p}_{1}p1​ 整除,不妨设为 q1{q}_{1}q1​ 。但 q1{q}_{1}q1​ 也是质数,因此 q1=p1{q}_{1} = {p}_{1}q1​=p1​ 。所以,比 nnn 小的正整数 n′=p2p3⋯pr{n}^{\prime } = {p}_{2}{p}_{3}\cdots {p}_{r}n′=p2​p3​⋯pr​ 也可以写成 q2q3⋯qs{q}_{2}{q}_{3}\cdots {q}_{s}q2​q3​⋯qs​ 。这与 nnn 的最小性矛盾！因此唯一性得证。

所以：任何一个合数n都可以分解为质因数的乘积，其中一定有一个最小的质因数（记为 p）。因此，只需要筛去质数的倍数，就能覆盖所有合数。剩下的未标记的数就是质数。

  1. 欧拉筛

关键在于，每个合数恰好会被它的最小质因子筛一次。

  1. Primes 中存的质数，是升序排列的。

  2. 所以当第一个质数能整除 i 时，它便是 i 的一个质因子，而且是最小的一个质因子。（e.g.3*7，3 一一定是 3*7 的最小质因子）

  3. 所以当第一次 i % primes[j] == 0 时，primes[j]是 i 的最小质因子，那么 primes[j]*i 的最小质因子一定是 primes[j]。

  4. Primes[j+1]*i 的时候，当 i % primes[j] == 0，primes[j+1]*i 的最小质因子不再是 primes[j+1]，而是 primes[j]，此时 primes[j+1]*i%primes[j]也一定为 0；不满足我们要说的每个合数恰好会被它的最小质因子筛一次。（如下图 3*4 的例子）
