еёёеҫ®еҲҶж–№зЁӢжҖ»з»“

1. дёҖйҳ¶еҫ®еҲҶж–№зЁӢ

1.1 еҸҜеҲҶзҰ»еҸҳйҮҸж–№зЁӢ

dydx=n(x)\frac{dy}{dx} = n(x)dxdy=n(x)

еҲҶзҰ»еҸҳйҮҸпјҢеҫ—еҲ°пјҡ

вҲ«dyg(y)=вҲ«f(x)dx+C\int \frac{dy}{g(y)} = \int f(x) dx + CвҲ«g(y)dy=вҲ«f(x)dx+C

1.2 йҪҗж¬Ўж–№зЁӢ

dydx=f(yx)\frac{dy}{dx} = f\left(\frac{y}{x}\right)dxdy=f(xy)

д»ЈжҚў y=vxy = vxy=vxпјҢеҢ–з®Җдёәпјҡ

dydx=v+xdvdxx\frac{dy}{dx} = \frac{v + x \frac{dv}{dx}}{x}dxdy=xv+xdxdv

дҫӢеҰӮпјҡ

dydx=y2x2вҲ’y2\frac{dy}{dx} = \frac{y^2}{x^2 - y^2}dxdy=x2вҲ’y2y2

йҖҡиҝҮеҸҳйҮҸд»ЈжҚўпјҡ

dudx=u21вҲ’u2\frac{du}{dx} = \frac{u^2}{1 - u^2}dxdu=1вҲ’u2u2

1.3 зәҝжҖ§ж–№зЁӢ

dydx+p(x)y=a(x)\frac{dy}{dx} + p(x) y = a(x)dxdy+p(x)y=a(x)

йҖҡи§Јдёәпјҡ

y=eвҲ’вҲ«p(x)dx(вҲ«a(x)eвҲ«p(x)dxdx+C)y = e^{-\int p(x)dx}\left( \int a(x) e^{\int p(x)dx} dx + C \right)y=eвҲ’вҲ«p(x)dx(вҲ«a(x)eвҲ«p(x)dxdx+C)

1.4 е…Ёеҫ®еҲҶж–№зЁӢ

M(x,y)dx+N(x,y)dy=0M(x, y) dx + N(x, y) dy = 0M(x,y)dx+N(x,y)dy=0

иӢҘж»Ўи¶іжқЎд»¶пјҡ

вҲӮMвҲӮy=вҲӮNвҲӮx\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}вҲӮyвҲӮM=вҲӮxвҲӮN

еҲҷж–№зЁӢеҸҜи§ЈгҖӮдҫӢеҰӮпјҡ

(1вҲ’2sinвҒЎ(y+x3))dx+(8xy3+x3y4)dy=0(1 - 2 \sin(y + x^3)) dx + (8x y^3 + x^3 y^4) dy = 0(1вҲ’2sin(y+x3))dx+(8xy3+x3y4)dy=0

и§Јжі•пјҡ

  1. зӣҙжҺҘжі•пјҡйҖҡиҝҮз§ҜеҲҶжұӮи§ЈгҖӮ
  2. з§ҜеҲҶеӣ еӯҗжі•пјҡжүҫеҲ°з§ҜеҲҶеӣ еӯҗпјҢдҪҝж–№зЁӢжҲҗдёәе…Ёеҫ®еҲҶж–№зЁӢгҖӮ
  3. еҸҳйҮҸд»ЈжҚўжі•пјҡйҖҡиҝҮеҸҳйҮҸд»ЈжҚўеҢ–з®Җж–№зЁӢгҖӮ

1.5 дјҜеҠӘеҲ©ж–№зЁӢ

dydx+p(x)y=a(x)yn\frac{dy}{dx} + p(x)y = a(x)y^ndxdy+p(x)y=a(x)yn

д»ЈжҚў z=y1вҲ’nz = y^{1-n}z=y1вҲ’n еҢ–дёәзәҝжҖ§ж–№зЁӢпјҡ

dzdx+(1вҲ’n)p(x)z=(1вҲ’n)a(x)\frac{dz}{dx} + (1-n)p(x)z = (1-n)a(x)dxdz+(1вҲ’n)p(x)z=(1вҲ’n)a(x)

2. й«ҳйҳ¶еҫ®еҲҶж–№зЁӢ

2.1 йҷҚйҳ¶жі•

  1. yвҖІвҖІ=yвҖІвҲ’yyвҖҷвҖҷ = yвҖҷ - yyвҖІвҖІ=yвҖІвҲ’y
  2. yвҖІвҖІ=f(x,y)yвҖҷвҖҷ = f(x, y)yвҖІвҖІ=f(x,y)
  3. yвҖІвҖІ=f(yвҖІ)yвҖҷвҖҷ = f(yвҖҷ)yвҖІвҖІ=f(yвҖІ)пјҢйҖҡиҝҮеҸҳйҮҸд»ЈжҚўз®ҖеҢ–дёәдёҖйҳ¶ж–№зЁӢгҖӮ
  4. f(x,y,yвҖІ)=0f(x, y, yвҖҷ) = 0f(x,y,yвҖІ)=0пјҢдҪҝз”ЁйҡҗеҮҪж•°жҲ–еҸӮж•°еҪўејҸи§ЈгҖӮ

2.2 зәҝжҖ§еҫ®еҲҶж–№зЁӢ

  1. yвҖІвҖІ=f(x)yвҖҷвҖҷ = f(x)yвҖІвҖІ=f(x)
  2. yвҖІвҖІ=f(x,y)yвҖҷвҖҷ = f(x, y)yвҖІвҖІ=f(x,y)
  3. yвҖІвҖІ=f(yвҖІ)yвҖҷвҖҷ = f(yвҖҷ)yвҖІвҖІ=f(yвҖІ)пјҢйҖҡиҝҮйҷҚйҳ¶еҸҳдёәдёҖйҳ¶ж–№зЁӢгҖӮ

дҫӢпјҡ

x2yвҖІвҖІ=L(yвҲ’xyвҖІвҲ’y2)x^2 yвҖҷвҖҷ = L(y - xyвҖҷ - y^2)x2yвҖІвҖІ=L(yвҲ’xyвҖІвҲ’y2)

3. зү№ж®Ҡзұ»еһӢзҡ„ж–№зЁӢ

3.1 еёёзі»ж•°зәҝжҖ§еҫ®еҲҶж–№зЁӢ

yвҖІвҖІ+pyвҖІ+qy=0yвҖҷвҖҷ + p yвҖҷ + q y = 0yвҖІвҖІ+pyвҖІ+qy=0

зү№еҫҒж–№зЁӢпјҡ

r2+pr+q=0r^2 + pr + q = 0r2+pr+q=0

и§Јдёәпјҡ

y=C1er1x+C2er2xy = C_1 e^{r_1 x} + C_2 e^{r_2 x}y=C1er1x+C2er2x

еҪ“ r1=r2r_1 = r_2r1=r2 ж—¶пјҢи§Јдёәпјҡ

y=(C1+C2x)er1xy = (C_1 + C_2 x) e^{r_1 x}y=(C1+C2x)er1x

3.2 еҸӮж•°еҸҳжҚўжі•

дҪҝз”ЁеҗҲйҖӮзҡ„еҸҳйҮҸжҲ–еҸӮж•°еҸҳжҚўпјҢз®ҖеҢ–жұӮи§Јж–№зЁӢгҖӮ

4. йқһйҪҗж¬Ўж–№зЁӢ

yвҖІвҖІ+p(x)yвҖІ+q(x)y=f(x)yвҖҷвҖҷ + p(x) yвҖҷ + q(x) y = f(x)yвҖІвҖІ+p(x)yвҖІ+q(x)y=f(x)

йҖҡи§Јдёәпјҡ

y=yh+ypy = y_h + y_py=yh+yp

е…¶дёӯ yhy_hyh жҳҜйҪҗж¬Ўж–№зЁӢзҡ„йҖҡи§ЈпјҢypy_pyp жҳҜзү№и§ЈгҖӮ

5. ж”№иҝӣж–№зЁӢзҡ„еҸҳжҚўж–№жі•

ж–№зЁӢеҪўејҸдёәпјҡ

x2yвҖІвҖІ+pxyвҖІ+qy=f(x)x^2 yвҖҷвҖҷ + p x yвҖҷ + q y = f(x)x2yвҖІвҖІ+pxyвҖІ+qy=f(x)
